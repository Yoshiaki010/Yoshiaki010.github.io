APIとは

「Application Program Interface」(アプリケーション・プログラム・インターフェース)
アプリケーションがOSにある各機能を使うための仕組み

CPUとは

「Central Processing Unit」(セントラル・プロセッシング・ユニット)
中央処理装置

CISCとは

「Complex Instruction Set Computer」(コンプレックス・インストラクション・セット・コンピュータ)
　CPUの命令体系の設計思想の一つで、1回の命令で複雑な処理ができる

DRAMとは

「Dynamic RAM」(ダイナミック・ラム)
コンデンサに蓄える電荷の状態で1ビットを表現する、SRAMに比べて大容量で安価

GUIとは

「Grahical User Interface」(グラフィカル・ユーザー・インターフェース)
アイコン、ボタン、チェックなどのように視覚的に操作するインターフェース

PWMとは

「PulseWideModulation」(パルスワイドモジュレーション)
パルス幅変調制御とも呼ばれ、モーターの回転速度やLEDの明るさなどをデジタル信号で制御する方式

OSSとは

「Open Source Softwere」(オープン・ソース・ソフトウェア)
ソースコードを公開しているソフトウェア
無保証が原則で、みんなで自由にソフトウェアを発展させていこうという考えがあるらしい

RFIDとは

「Radio Frequency IDentification」(ラディオ・フレキュエンシー・アイデンティフィケーション)
小さいICチップにアンテナがついている荷札電子、ICタグとも呼ばれる

USBとは

「Universal Serial Bus」(ユニバーサル・シリアル・バス)
周辺機器の接続したり、スマホやタブレットの充電したり、データを転送したりするために使われる
